# MyGuiApp

## Prerequisite

```bash
sudo apt-get install libgirepository1.0-dev    # For Ubuntu
sudo dnf install gobject-introspection-devel   # For Fedora

sudo apt-get install libgtk-3-dev
sudo dnf install gtk3-devel

sudo apt-get install libgraphene-1.0-dev
sudo dnf install graphene-devel

sudo apt-get install libgtk-4-dev libadwaita-1-dev
sudo dnf install gtk4-devel libadwaita-devel
```
